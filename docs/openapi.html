Backoffice Timesheet Importer API (1.0.0)

Download OpenAPI specification:

License: Proprietary - Internal Use Only

API del microservizio Backoffice per l'import dei timesheet Shifts e la gestione dei dipendenti (Worker).

Note: l'autenticazione è fuori scope di questo servizio. Gli endpoint sono pensati per uso interno al Backoffice.

Avvia l'import di un timesheet

Riceve l'identificativo del file Excel precedentemente caricato su aml.file. Crea un ImportBatch in stato queued e avvia l'elaborazione asincrona. Il servizio non modifica mai i timestamp originali: le pause mal registrate sono derivate come entità Pause e associate ai turni con regola di overlap massimo.

Request Body schema: application/json
required
fileID
required
string <uuid>

Identifier returned by aml.file for the uploaded spreadsheet.

Responses

https://backoffice.flowpay.local/imports/timesheet
